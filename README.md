# OpencartV121 - Automated Testing Framework

Automated end-to-end testing suite for OpenCart v1.2.1 ecommerce platform using Selenium WebDriver, TestNG, and Java.

## 📘 Project Overview

This project is a comprehensive test automation framework designed to validate the functionality of OpenCart v1.2.1, a popular open-source ecommerce platform. As a QA Automation Engineer, this framework simulates real-world testing scenarios across multiple browsers and environments, ensuring product quality and reliability through continuous integration.

---

## 🎯 Project Objectives

✅ Automate end-to-end testing for OpenCart v1.2.1  
✅ Support cross-browser testing (Chrome, Firefox, Safari, Edge)  
✅ Validate critical user workflows (Login, Product Search, Cart Management, Checkout)  
✅ Generate comprehensive test reports and screenshots  
✅ Enable continuous integration and CI/CD pipeline integration  
✅ Maintain data-driven testing approach with parameterized test cases
